# Typecho-SlideShow
根据typecho的热门文章用滑动幻灯片的形式在相应页面展示出来

# 原理
是基于TePostView插件生成热门文章，然后整合carousel.min.js等达到幻灯片滑动播放的效果。

# 食用方法
在相应的页面/首页需要展示的地方填上 `<?php SlideShow_Plugin::outputSlideShow() ?> `即可
